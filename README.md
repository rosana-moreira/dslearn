# Como executar o projeto

## Back end
Pré-requisitos: Java 11+

```bash
# clonar repositório
git@github.com:rosana-moreira/dslearn.git
# entre na pasta da aplicação
cd dslearn
# executar o projeto
./mvnw spring-boot:run
```
